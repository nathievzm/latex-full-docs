\begin{document}

We use the \texttt{amsmath} package command
\verb|\text{...}| to create text-only fractions
like this:

\[\frac{\text{numerator}}{\text{denominator}}\]

Without the \verb|\text{...}| command the result
looks like this:

\[\frac{numerator}{denominator}\]

\end{document}
